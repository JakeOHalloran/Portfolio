Goodreads Web Scraper
ABOUT
TECHNOLOGY
LINKS
An author I was creating a website for asked if there was a way for the website to pull his books from other sources, so as to not need a CMS. The website Goodreads stored all his books, but had an API that left a lot to be desired. So I created a web scraper that would pull books from Goodreads. I created a simple frontend to showcase this web scraper.
The web scraper was created with PHP, using simple_html_dom to parse the web pages. The front end was created using Angular 11, Bootstrap and Sass.
The web app can be used here. The source code can be browsed on github

Game Backlog
ABOUT
TECHNOLOGY
LINKS
This app uses the IGDB api to allow for the creation of a video game backlog to track what games you want to play but haven't gotten around to. Users can log in and edit their backlog (filter by platform, remove games, mark as completed), or add to it by searching games by their title. The app takes the search string and returns a list of games like it from the IGDBs game database. These results can be added to the users backlog.
This app was created using Angular 11 with Bootstrap and Sass, and NodeJS with a MySQL database.
The source code can be browsed on github. The web app can be used here. To log in, use the details 'demo@gamebacklog.com' with the password 'demo_password'.
Automated Negotiation System

For my masters I investigated using automated negotiation systems within an ecommerce store. Could it be used to enable flexible pricing? Could store owners rely on this system to negotiate with customers on their behalf? To answer these questions I created my own automated negotiation system. This system uses previous negotiations to learn how to get better prices from customers.
The negotiation system itself was created using PHP and MySQL. To test it within an ecommerce setting, I created a basic store front using Angular 6 and Bootstrap.
